🚀 QUICK START
═══════════════════════════════════════════════════════════════════════════

1. Read the Docs (5 min)
   → Open: QUICK_START.md

2. Install Dependencies (1 min)
   → cd newstore-ordering-app
   → flutter pub get

3. Run the App (1 min)
   → flutter run -d chrome

4. Test the App (5 min)
   → Click "Continue as Guest"
   → Explore all screens
   → Try adding stores/vendors/products

5. Set Up Firebase (10 min)
   → Create Firebase project
   → Enable Firestore
   → Configure authentication

6. Deploy (Optional)
   → flutter build web --release
   → firebase deploy --only hosting
